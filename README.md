Créez un site avec 4 pages différentes (Index, S’enregistrer, Profile et Login). La page profile on doit
être connecté pour la voir. Essayez de créer une base de données qui aura une table utilisateurs
avec un id, nom, prénom, pseudo, un mot de passe et un âge. Créez une table villes avec un id,
nom, pays et capitale qui contient une liste de 10 villes (relier les 2 tables). Créer une page
s’enregistrer qui demande à l’user de s’enregistre dans la table. Ensuite créer une page login qui
demande un pseudo et un mot de passe.

Personne 1
Page Index et s'enregistrer. Création de la base de données et de la table villes + créer les 10 villes.
Dans la page index je veux voir une vidéo et un tableau contenant les 10 pays en base de données.

Personne 2
Page Profile et login. Créer la table utilisateurs en base de données. Faire la relation en base de
données entre les 2 tables. Attention le champ que vous utiliserez pour la clé secondaire, il peut être
null. Dans profile je veux voir les informations de la personne qui s'est connecté (Nom, prénom et
pseudo). Et mettez une image.

Créez 3 utilisateurs via le formulaire d’inscription bien entendu.

Dans le code PHP, je veux que vous utilisiez la programmation orientée objet. Donc je veux une
classe Utilisateur.php et une classe Ville.php. Attention il y a une relation entre ces 2 classes, ça
implique des choses à rajouter.

1. Faites en sortes que quand on s'enregistre, il nous demande de quelle ville on est. Sous forme de
   liste à choisir en fonction des villes qu'on a en base de données.

2. Faites en sorte que dans profile, on voit les informations de l'utilisateur qui s'est connecté en
   donnant maintenant sa ville et ATTENTION sa nationalité. C'est à dire si sa ville c'est Bruxelles donc
   son pays c'est Belgique, il faut qu'on voit qu’il est Belge. Je ne veux donc pas voir le nom du pays
   mais plutôt sa nationalité.
3. Je rappelle que vous pouvez utiliser toutes vos connaissances en front end pour rendre plus beau
   votre site. N'oubliez jamais que vous travaillez pour vous, c'est votre expérience qui vous rendra plus
   compétent.
